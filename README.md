A website for displaying **Mechanical Keyboards** database. Inspired by [GSMArena](https://www.gsmarena.com/).

## [Visit Here 🚀](http://frasnym.tech)

## Features
- ✍ Written in TypeScript
- 💻 Builded with NextJs
- 🚀 Deployed on Vercel

## How to run

- Without docker
```**bash**
npm run dev
```
